# Spur Chat App

This is a full-stack AI chat application built with **Next.js**, **PostgreSQL**, **Prisma**, and **LLMs**, inspired by ChatGPT.

---

## What is this project?

Spur Chat is a real-time chat app where users can:
- Authenticate using OAuth
- Create multiple chats
- Send messages with instant (optimistic) UI updates
- Receive streamed AI responses

The app focuses on clean architecture, fast UX, and modern React patterns.

---

## Tech Stack

- **Next.js (App Router)**
- **NextAuth**
- **Prisma**
- **PostgreSQL (Docker)**
- **React Query**
- **Zustand**
- **Tailwind CSS**
- **Framer Motion**
- **LLMs via OpenRouter / OpenAI / Anthropic**
